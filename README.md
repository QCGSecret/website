## Development

Requirements:

- [Node.js](https://nodejs.org/en/download/)
- [pnpm](https://pnpm.io/installation)

Running the development server:

```bash
pnpm install
pnpm dev
```

## Deployment

The production build is ran with [Docker](https://www.docker.com/).

```bash
docker compose up -d --build --remove-orphans
```
